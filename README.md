# Streamlit Restaurant Classifier

This app fetches restaurants near a Google Maps location and classifies them step-by-step using OpenAI ChatGPT (gpt-3.5-turbo).

## Setup on Streamlit Cloud

1. Fork / clone this repo.
2. Add your API keys in **Secrets**:
   - `GOOGLE_MAPS_KEY`
   - `OPENAI_API_KEY`
3. Ensure `requirements.txt` is present.
4. Deploy on Streamlit Cloud: https://streamlit.io/cloud
